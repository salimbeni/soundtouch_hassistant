# Changelog

## 1.3.1
- Improved artwork detection when saving presets (now uses high-quality logos if available).

## 1.3.0
- **Long-Press to Save**: Save the current station to a preset by holding the preset button (0.8s).
- Visual feedback and animations during preset saving.

## 1.2.2
- Fixed raw HTML display in "empty favorites" message.

## 1.2.1
- Added `CHANGELOG.md` support for Home Assistant.

## 1.2.0
- **Responsive Fluid Layout**: The app now fills the entire window in Home Assistant.
- Optimized UI for desktop and tablet views (up to 6 columns for presets).
- Centered and scaled player controls for better accessibility on large screens.

## 1.1.0
- Added **Home Assistant Ingress** support.
- Implemented persistent storage in Home Assistant's `/data` directory.
- Updated documentation and installation guides.

## 1.0.0
- Initial release for Home Assistant.
- Integrated Guided Start Wizard.
- Modernized Dark Theme with glassmorphism.
- Personalization features and greetings.
